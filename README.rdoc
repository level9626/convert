== README

Hello.

Application allow you to convert 2 types of files (JSON & XML) in CSV format and download converted file immediately.

== VISIT WORKABLE VERSION
https://mighty-anchorage-8264.herokuapp.com

== OR SETUP LOCALLY
- install Postgresql
- install gems bundle
- setup database rake db:create && rake db:migrate
- run the server <tt>rails s</tt>
- run tests <tt>rake test</tt>
